# SpringBoot整合Druid

## 返回 [README.md](../../README.md)

## 1.引入依赖
```xml
<!-- 使用这个替代druid 可以简化druid配置，application.yml中可以用spring.datasource.druid.xxx直接配置 -->
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>druid-spring-boot-starter</artifactId>
    <version>${druid.version}</version>
</dependency>
```
